🎓 Academic Management System

Sistema acadêmico desenvolvido em Python com integração a banco de dados, projetado para simular um aplicativo de gerenciamento universitário.

Este projeto foi criado com o objetivo de aplicar conceitos de programação, organização de sistemas e manipulação de dados em um ambiente acadêmico.

📌 Sobre o projeto

O Academic Management System permite o gerenciamento de informações acadêmicas, como alunos, disciplinas e notas, simulando funcionalidades básicas de sistemas utilizados em instituições de ensino.

Este projeto faz parte do meu processo de aprendizado em desenvolvimento de software utilizando Python.

⚙️ Funcionalidades

✔ Cadastro de alunos
✔ Gerenciamento de disciplinas
✔ Registro de notas
✔ Consulta de informações acadêmicas
✔ Atualização de dados
✔ Persistência de dados em banco de dados

🛠 Tecnologias utilizadas

Python 3

Banco de Dados SQL

CRUD (Create, Read, Update, Delete)

Git

GitHub
